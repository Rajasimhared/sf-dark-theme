## Installation

### From Source

1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" (toggle in top right)
4. Click "Load unpacked"
5. Select the extension folder
6. Done! Click the extension icon to toggle dark mode

## Usage

1. Click the extension icon in your Chrome toolbar
2. Toggle the switch to enable or disable dark mode
3. Your preference is saved automatically

## Technical Details

- Uses CSS filter inversion with optimized hue rotation
- Applies to main documents, iframes, and dynamically loaded content
- Minimal performance impact
- Works with Chrome Manifest V3

## Privacy

This extension does NOT collect any data. It only stores your dark mode preference locally. See [PRIVACY.md](PRIVACY.md) for details.
