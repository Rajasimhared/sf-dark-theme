# Chrome Web Store Listing

## Name
Dark Inverter

## Short Description (132 characters max)
Invert any website's colors to dark mode instantly. Eye-friendly, fast, and private. Perfect for night browsing and reducing strain.

## Detailed Description

Transform any website into a comfortable dark mode experience with just one click!

**Features:**
✓ Instant dark mode for any website
✓ Eye-friendly color inversion optimized for reading
✓ Smart handling of images and videos (they stay normal!)
✓ Works on iframes and dynamic content
✓ Persistent preferences across all tabs
✓ Lightweight and fast
✓ 100% private - no data collection

**How it works:**
Simply click the extension icon and toggle dark mode on or off. Your preference is saved automatically and applies to all websites.

**Perfect for:**
- Late-night browsing
- Reducing eye strain
- Battery saving on OLED screens
- Anyone who prefers dark themes

**Privacy:**
This extension does NOT collect, store, or transmit any personal data. It only saves your dark mode preference locally on your device.

**Open Source:**
The source code is available on GitHub. Feel free to review, contribute, or fork!

Made with ❤️ for dark mode enthusiasts.
